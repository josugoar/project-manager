FAQ

What is Project Manager?

Project Manager is a web-based application that helps you manage your projects and tasks. You can create, edit, and delete projects, assign resources, track progress, and communicate with your team and clients.

How do I create a new project?

To create a new project, you need to go to the New Project page and fill out the project registration form. You need to provide the project title, start and end dates, person responsible, description, client information, and project type. Once you submit the form, your project will be created and added to the list of projects.

How do I view the details of a project?

To view the details of a project, you need to go to the Projects page and click on the View Details link of the project you want to see. You will be taken to the Project Details page, where you can see all the information about the project, such as start and end dates, client data, description, resources involved, planned tasks, comments, etc.

How do I edit or delete a project?

To edit or delete a project, you need to go to the Project Details page and click on the Edit or Delete button at the top right corner of the page. You will be able to modify or remove the project information as you wish. Please note that deleting a project is irreversible and will erase all the data associated with it.

How do I add or remove resources from a project?

To add or remove resources from a project, you need to go to the Project Details page and click on the Resources tab. You will see a list of resources involved in the project, as well as a dropdown menu to select new resources. You can add or remove resources by checking or unchecking the boxes next to their names. You can also assign different roles and responsibilities to each resource by clicking on the Edit button next to their name.

How do I create or update tasks for a project?

To create or update tasks for a project, you need to go to the Project Details page and click on the Tasks tab. You will see a list of planned tasks for the project, as well as a button to add new tasks. You can create or update tasks by filling out the task form, which includes the task title, description, deadline, status, and assigned resource. You can also delete tasks by clicking on the Delete button next to each task.

How do I add or view comments for a project?

To add or view comments for a project, you need to go to the Project Details page and click on the Comments tab. You will see a list of comments for the project, as well as a textbox to write new comments. You can add or view comments by typing in the textbox and clicking on the Submit button. You can also reply to or delete comments by clicking on the Reply or Delete button next to each comment.
